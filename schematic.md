# Электрическая схема контроллера сварочного аппарата

## Архитектура полумоста инвертора

```
              +380V (DC)
                |
              +---+
              |   |
             Q1   R_top (защитный резистор)
              |   |
              +---+
                |
      +-----+---+---+-----+
      |     |       |     |
    CCP1  GND    OUT1   GND
      |                  
      |              (Выход инвертора)
      +-----+---+---+-----+
                |
              +---+
              |   |
             Q2   R_bot (защитный резистор)
              |   |
              +---+
                |
              -GND (DC)
```

## Разводка выводов PIC18F4550

### Выводы питания
- **VDD (pin 11, 32)**: +5V (регуляторное напряжение)
- **VSS (pin 12, 31)**: GND
- **VUSB (pin 19)**: +3.3V (если используется USB)

### PWM выводы (управление ключами)
- **RC2 (pin 17)**: CCP1 → Q1 (верхний ключ полумоста)
- **RC1 (pin 16)**: CCP2 → Q2 (нижний ключ полумоста)

### Аналоговые входы (датчики)
- **RA0 (pin 2)**: AN0 - датчик сварочного тока
- **RA1 (pin 3)**: AN1 - датчик сетевого напряжения
- **RA2 (pin 4)**: AN2 - датчик температуры диодов

### Цифровые выходы (управление)
- **RB0 (pin 21)**: Выход мягкого старта
- **RB1 (pin 22)**: LED индикатор ошибки

### UART (диагностика)
- **RC6 (pin 18)**: TX (передача данных)
- **RC7 (pin 19)**: RX (прием данных)

### Кварцевый генератор
- **OSC1 (pin 13)**: Входной сигнал от кварца (20 МГц)
- **OSC2 (pin 14)**: Выходной сигнал кварца
- **C_in = 10pF**, **C_out = 10pF** (конденсаторы кварца)

## Схема подключения датчиков

### Датчик сварочного тока (AN0)

```
Датчик тока           PIC18F4550
(Холловский или шунт)  
         0-5V ┌────────────────┐
              │ RA0/AN0        │
              │                │
         GND ─┴────────────────┘
```

**Калибровка:**
- 0V → 0A
- 5V → 100A
- Преобразование: I(A) = (ADC / 1024) × 100

### Датчик сетевого напряжения (AN1)

```
Сетевое напряжение
220V AC          Выпрямитель      Делитель напряжения
                                   
    ┌─────────────────────────────┐
    │                             │
   AC        Диодный мост      Фильтр
    │        (230V DC)            │
    └─────────────────────────────┘
                                   R_top = 100k
                                      |
                                      +── 0-5V → RA1/AN1
                                      |
                                   R_bot = 20k
                                      |
                                     GND
```

**Расчет делителя:**
- U_out = U_in × R_bot / (R_top + R_bot)
- U_out = 230V × 20k / 120k ≈ 38.3V (на 230V AC)
- **Необходимо через выпрямитель до ~320V DC → 5V**
- Используйте R_top = 150k, R_bot = 10k

**Калибровка:**
- 0V → 0V
- 5V → 300V
- Преобразование: U(V) = (ADC / 1024) × 300

### Датчик температуры (AN2) - LM35

```
LM35 датчик температуры

    +5V
     │
     ├─────────┐
     │       100nF
     │         │
   [●] LM35    ├──── RA2/AN2 (0-5V)
     │         │
     └─────────┴──── GND

Выход: 10 mV/°C
0V = 0°C
1V = 100°C
5V = 500°C (ограничено 120°C в коде)
```

**Калибровка:**
- T(°C) = (ADC / 1024) × 120

## Управление ШИМ

### Драйвер затвора (Gate Driver)

```
PIC (CCP1)                      Силовые ключи
    │
    │ 0-5V                     MOSFET Q1
    └──────[100Ω]──────┬───────>[●]───┐
                       │               ├─ +380V DC
                   Gate Driver         │
                   (IR2110 или         │
                    MIC4427)           │
                       │               ├─ -GND DC
    ┌──────[100Ω]──────┤───────>[●]───┘
    │ 0-5V             │       MOSFET Q2
PIC (CCP2)            │
                   Ground ─────────────┘
```

**Параметры ШИМ:**
- **Частота**: 16 кГц
- **Мертвое время**: ~500 нс (между включением Q1 и Q2)
- **Макс. ток затвора**: 2A
- **Макс. напряжение затвора**: 20V

## Мягкий старт (RB0)

```
PIC (RB0)           Управление мягким стартом
   │
   │ 0-5V Output
   └──────[1kΩ]──────┐
                      ├───→ TRIAC контроль напряжения
                      │     на первичной обмотке
                 Opto ├───→ Импульс каждые 100 мс
                 Coupler
                      │
                    GND
```

**Программный мягкий старт:**
1. Начальный ШИМ: 0%
2. За 1 сек увеличить ШИМ от 0% до 100%
3. Ток наращивается плавно, избегая пиковых токов

## Защитные резисторы и конденсаторы

### RC фильтр на аналоговых входах

```
Датчик          PIC
   │
   ├─[10kΩ]──┬──→ RA0/AN0
   │         │
   │      [100nF]
   │         │
   └─────────┴──── GND
```

**Назначение:**
- Снижение шумов с датчиков
- Защита входа АЦП
- Постоянная времени τ = R×C = 10k × 100n = 1ms

### Конденсаторы развязки питания

```
+5V
  │
  ├─[100nF]─┬──→ VDD (PIC)
  │         │
  │      [10μF]
  │         │
  └─────────┴──── GND
```

## LED индикатор ошибки (RB1)

```
PIC (RB1)          LED
   │ 0-5V Output
   │
   ├──[330Ω]──[●]──┐
   │                │
   └────────────────┤ -GND
                    │
                  Красный LED
```

**Функция:**
- Загорается при любой ошибке защиты
- Мигает с частотой 1 Гц при критической ошибке

## Питание контроллера

```
+380V DC             +5V Регулятор        PIC18F4550
 (из инвертора)          (AMS1117-5)
     │                        │
     ├─[220μF/450V]──→ IN ──→ OUT ────→ VDD
     │                        │
     └────────────────────────┴─ GND ──→ VSS
```

## Протокол UART для отладки

```
ПК ↔ USB-UART ↔ PIC18F4550
   (9600, 8N1)

Провода:
TX → RC6 (PIC TX)
RX → RC7 (PIC RX)
GND → GND
```

## Полная простая блок-схема

```
┌─────────────────────────────────────┐
│     Сетевое напряжение 220V AC      │
└────────┬────────────────────────────┘
         │
         ├─→ Выпрямитель + Фильтр (380V DC)
         │
         ├─→ Инвертор (полумост)
         │    ├─→ ШИМ от CCP1 → Q1
         │    └─→ ШИМ от CCP2 → Q2
         │
         └─→ Вторичная обмотка (с��арка)

┌─────────────────────────────────────┐
│      Датчики контроля               │
├─────────────────────────────────────┤
│ • Ток: AN0 (RA0)                    │
│ • Напряжение: AN1 (RA1)             │
│ • Температура: AN2 (RA2)            │
└────────┬────────────────────────────┘
         │
         └─→ PIC18F4550
              ├─→ ПИД управление ШИМ
              ├─→ Защита (перегрузка, перегрев)
              ├─→ Мягкий старт (RB0)
              ├─→ LED ошибка (RB1)
              └─→ UART диагностика (RC6/RC7)
```

## Рекомендации по монтажу

1. **Разделение цепей**: Высоковольтная часть (силовая) отдельно от низковольтной (управления)
2. **Заземление**: Звездообразная топология GND для предотвращения шумов
3. **Развязка питания**: Конденсаторы 100nF/10μF как можно ближе к PIC
4. **Экранирование**: Провода датчиков в экран, если возможно
5. **Кварц**: Конденсаторы 10pF рядом с PIC, провода как можно короче
