# PIC Welder Controller

Программа управления инверторным сварочным аппаратом на микроконтроллере PIC18F4550 с архитектурой полумоста.

## Характеристики

### Микроконтроллер
- **PIC18F4550** (48 МГц)
- Встроенный кварцевый генератор 20 МГц + PLL
- 8 канальный 10-битный АЦП
- 2 модуля PWM (CCP1, CCP2)
- UART для диагностики

### Управление сваркой
- **Архитектура инвертора**: Полумост (2 ключа - Q1, Q2)
- **ШИМ частота**: 16 кГц
- **Управление ШИМ**: ПИД-регулятор для стабилизации сварочного тока
- **Диапазон тока**: 0-100 А
- **Управление мягким стартом**: Плавный разгон за 1 секунду

### Датчики и защита
- **Датчик тока**: AN0 (RA0) - диапазон 0-100 ��
- **Датчик напряжения**: AN1 (RA1) - диапазон 0-300 В
- **Датчик температуры**: AN2 (RA2) - диодные силовые компоненты

### Функции защиты
- ✅ Контроль завышенного напряжения (> 280 В)
- ✅ Контроль заниженного напряжения (< 160 В)
- ✅ Контроль перегрузки по сварочному току (> 110 А)
- ✅ Контроль перегрева диодов (> 85 °C)
- ✅ Индикация ошибок через LED (RB1)
- ✅ Отключение ШИМ при любой ошибке

### Диагностика
- **UART**: 9600 бод, 8 бит, 1 стоп-бит
- **Вывод диагностики**: каждые 500 мс
- **Формат**: `DATA: I=45A U=220V T=65C PWM=150 STATE=1 CYCLES=2451`

## Команды UART

| Команда | Описание |
|---------|----------|
| `S` | Включить сварку (запустить мягкий старт) |
| `P` | Выключить сварку |
| `+` | Увеличить целевой ток на 5 А |
| `-` | Уменьшить целевой ток на 5 А |
| `?` | Показать справку по командам |

## Параметры защиты

**Все параметры можно настроить в файле `config.h`:**

```c
#define MAX_WELDING_CURRENT 100     // Макс. ток сварки (А)
#define MAX_GRID_VOLTAGE 280        // Макс. сетевое напряжение (В)
#define MIN_GRID_VOLTAGE 160        // Мин. сетевое напряжение (В)
#define MAX_DIODE_TEMP 85           // Макс. температура диодов (°C)
#define OVERCURRENT_THRESHOLD 110   // Порог перегрузки по току (A)
#define SOFT_START_TIME 1000        // Время разгона (мс)
#define PWM_FREQUENCY 16000         // Частота ШИМ (Гц)
```

## Компиляция и загрузка

### Требования
- **MPLAB X IDE** v5.40 или выше
- **XC8 Compiler** v2.30 или выше
- **PICKit3** или аналогичный программатор

### Шаги
1. Откройте проект в MPLAB X IDE
2. Выберите конфигурацию: `Debug` или `Release`
3. Нажмите "Clean and Build Main Project"
4. Подключите программатор
5. Нажмите "Program Device"

## ПИД-регулятор

Сварочный ток регулируется ПИД-регулятором:

```c
typedef struct {
    int16_t error;              // Ошибка
    int16_t last_error;         // Предыдущая ошибка
    int16_t integral;           // Интегральная часть
    uint8_t kp;                 // P коэффициент = 2
    uint8_t ki;                 // I коэффициент = 0
    uint8_t kd;                 // D коэффициент = 1
} PIDController;
```

**Настройка коэффициентов** в `main.c`:
- Увеличьте `kp` для быстрой реакции на ошибку
- Увеличьте `ki` для устранения ошибок смещения
- Увеличьте `kd` для демпфирования колебаний

## Состояния системы

| State | Описание |
|-------|----------|
| 0 | IDLE - ожидание команды |
| 1 | RUNNING - мягкий старт или сварка активна |
| 2 | ERROR - обнаружена ошибка защиты |
